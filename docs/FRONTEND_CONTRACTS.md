Objetivo
Garantizar la "Verdad Operacional" en la interfaz de usuario de WordPress.

Contexto
Widget flotante para captura de leads.

Alcance permitido
HTML, CSS (aislado), JavaScript vainilla o React ligero.

Reglas técnicas obligatorias
Offset: 85px desde el borde inferior.


Timeouts: Toda petición de red debe abortar a los 15s con mensaje de error claro.


Retroalimentación: Usar Skeletons para la carga inicial del chat.

Criterios de aceptación
Visualización correcta en móviles (Safari/Chrome) sin tapar el botón de WhatsApp.
